== Description ==

This plugin will significantly increase your adsense revenue. It shows google adsense ad on left side of your blog, due to prominent position it maximizes the CTR and revenue of your site. use only 160x600 vertical ad format
You can also use it with any other ad network.

This plugin will show one author's ad per hundred impressions. By using this plugin you consent to display author's ad on your site.

see plugin in action
http://www.ahsan.pk

= Features =

It has only one feature. That is it will markedly increase your adsense revenue.

== Installation ==



1. Upload complete plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the Settings -> AdSense Float and enter your AdSense code.

You can use advertising code of any network with this plugin.

= Can I use other network ads instead of adsense? =

Yes. You can use this plugin to show ads from any ad network you like.
Just paste the code in the box and it will work fine. Please use only vertical ads.

= How Adsense Float increases revenue? =

It places the ad on left side of your site, ad scrolls down with content, due to its prominent position it increases the CTR.
So you get more revenue.
